# student_gym
<h1>器材租借系統</h1>
<hr>

<h3>專案內容</h3>
<p>介紹：發想自工作上想解決的需求，學生透過手寫的方式登記想要使用的器材，想藉由建構網站讓同學能夠自行登記並且歸還，管理器材的人員也能夠藉由查詢看到同學的使用時間以及借用了哪些器材，不須再透過管理人員才能借器材</p> 

 <h3>使用技術</h3>
<li>MVC架構</li>
<li>JDBC</li>
<li>JSP/Servlet</li>
<li>HTML</li>
<li>MySQL</li>
